一.数据说明
注：请将所有.csv文件放在同一目录下
1.交易记录.csv：记录所有的基金购买信息。
特别说明：v2表中‘采样周期’已替换为'交易时间'，相应代码也已作修改。（C1~C5.csv中该列相同）
2.当前用户交易记录.csv：针对当前需要推荐的客户，将其交易信息写入该文件，聚类程序会读取该文件。
3.用户记录.csv:以单个客户为单位处理交易数据，计算有关指标并得到了其所属类别。
4.判别公式参数.csv：由聚类程序自身读取与修改。
5.C1～C5.csv：将分类后的客户的交易记录写入文件，同一文件涵盖的客户属于同一类别（该些文件在聚类时处理较麻烦，等最后和推荐算法合并时考虑是否保留，目前代码中未修改该些文件）。
