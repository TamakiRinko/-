*********************FundList.py************************
用于爬取天天基金网的基金数据，这里我将所有基金代码以及类型爬取了下来，存储在基金列表.csv中
********************Depart.py****************************
将基金列表.csv拆分成四类基金：股票，混合，债券，货币，分别存在
StockList.csv
AllocList.csv
BondList.csv
CurrList.csv
*********************CreateData.py************************
基于四类基金生成训练集

其中各种比例权重完全基于论文

可以改变的变量有：
UserNum				(用户总数)
各种weightlist			(权重)
各种start/end			(精度范围) 
最终结果为TrainData.csv

注意：每次结果必定不相同，若要进行训练与检查，请选定一次结果不要改变。
